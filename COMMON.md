🎨 Core Theme (Agri + Ecommerce Feel)
✅ Primary (Your Brand)
Green 600 → #2E7D32 (main brand color)
Green 500 → #4CAF50 (buttons, highlights)
Green 100 → #C8E6C9 (background accents)

👉 Use darker green for trust, lighter for UI softness

✅ Secondary (Earth / Farmer Identity)
Soil Brown → #8D6E63
Light Brown → #D7CCC8

👉 Adds “farm authenticity” (don’t overuse)

✅ Accent (Call-to-Action / Ecommerce feel)
Orange (CTA) → #FF9800
Dark Orange → #F57C00

👉 Use ONLY for:

Buy buttons
Offers
Discounts
✅ Neutral (Clean UI)
White → #FFFFFF
Light Gray → #F5F5F5
Border Gray → #E0E0E0
Text Dark → #000000ff
Text Medium → #000000ff
✅ Status Colors
Success → #4CAF50 (same as green)
Error → #F44336
Warning → #FFC107
Info → #2196F3
🧩 Usage Rules (Important)

Don’t mix randomly:

🟢 Green → branding, headers, primary buttons
🟠 Orange → ONLY actions (Buy Now, Checkout)
🤍 White/Gray → keep UI clean like Amazon
🤎 Brown → minimal (icons, badges, farmer tags)
🔥 Example UI Mapping
Header → Dark Green
“Add to Cart” → Green
“Buy Now” → Orange
Cards → White + light shadow
Background → Light gray
⚠️ Common Mistake

If everything is green:
→ No hierarchy
→ Looks cheap
→ Users don’t know where to click
